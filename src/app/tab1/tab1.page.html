BIENVENIDO A IONIC

Ionic proporciona un conjunto de herramientas para crear aplicaciones nativas de iOS y Android, así como aplicaciones web progresivas para dispositivos móviles, usando bibliotecas web, marcos y lenguajes familiares.

Ionic framework, que es gratis y de código abierto, ofrece una biblioteca de componentes, gestos y herramientas de interfaz de usuario optimizados para dispositivos móviles.
